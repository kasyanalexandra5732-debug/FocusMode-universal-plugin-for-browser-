# 🎯 FocusMode: Productivity Timer Extension

[![Manifest V3](https://img.shields.io/badge/Manifest-V3-blue.svg)](https://developer.chrome.com/docs/extensions/mv3/intro/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**FocusMode** — это мощное браузерное расширение для Chromium, созданное для борьбы с цифровой прокрастинацией. Оно блокирует доступ к отвлекающим ресурсам на сетевом уровне, помогая вам сохранять концентрацию.

---

## 🚀 Основной функционал

* **⚡ Динамический «Черный список»**: Добавляйте и удаляйте любые домены (например, `vk.com`, `youtube.com`) "на лету" через интерфейс.
* **⏳ Таймер обратного отсчета**: Установка времени глубокого фокуса от 1 до 120 минут.
* **🤖 Автоматическая разблокировка**: Благодаря `chrome.alarms`, доступ к сайтам вернется ровно в срок, даже если браузер был закрыт или расширение выгружено из памяти.
* **🛡️ Нативная сетевая блокировка**: Использование `declarativeNetRequest` гарантирует, что запросы к сайтам будут обрезаны еще до начала загрузки страницы (экономия трафика и высокая скорость).
* **💾 Сохранение состояния**: Список сайтов и статус таймера надежно хранятся в `chrome.storage.local`.
* **🔒 Умный интерфейс**: Во время работы таймера функции редактирования списка блокируются, чтобы исключить возможность "читинга".

## 📥 Инструкция по установке (Chromium-браузеры)

Данная инструкция является универсальной и подходит для **Google Chrome, Яндекс.Браузера, Microsoft Edge, Opera и Brave**.

1.  **Подготовьте файлы** 📁
    * Убедитесь, что все файлы расширения (`manifest.json`, `background.js`, `popup.html` и др.) находятся в одной локальной папке на вашем компьютере (например, в папке `focus-mode`).

2.  **Откройте страницу расширений** 🌐
    * Введите в адресную строку браузера: 
      * Для Chrome/Brave/Yandex: `chrome://extensions/`
      * Для Microsoft Edge: `edge://extensions/`

3.  **Включите режим разработчика** 🛠️
    * Найдите переключатель **Developer mode** (Режим разработчика) в правом верхнем углу и активируйте его.

4.  **Загрузите расширение** 📤
    * Нажмите на появившуюся кнопку **Load unpacked** (Загрузить распакованное расширение).
    * В системном диалоговом окне выберите папку `focus-mode`, в которой лежат файлы проекта.

5.  **Закрепите для быстрого доступа** 📌
    * Нажмите на иконку «пазла» (Расширения) в панели инструментов браузера.
    * Найдите в списке **Focus Mode** и нажмите на иконку булавки (**Pin**), чтобы таймер всегда был на виду.
